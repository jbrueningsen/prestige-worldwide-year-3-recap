## Publish on GitHub Pages
1. Commit `index.html` and the `assets/` folder to the repo root.
2. Repo → Settings → Pages
3. Build and deployment: **Deploy from a branch**
4. Branch: `main` / folder: `/ (root)`

Then open:
https://<username>.github.io/<repo>/
